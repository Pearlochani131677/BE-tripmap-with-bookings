Features:
- Group travel
- Budget-based planning
- Stamina-aware scheduling
- Real attraction search
- Hourly itinerary in cards
- Memory system
- Google Maps embed

Setup:
1. Get free API key: https://opentripmap.io
2. Backend:
   cd backend
   npm install
   node server.js
3. Frontend:
   cd frontend
   npm install
   npm run dev
